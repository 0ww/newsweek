### Prerequisites

- Node.js (for Browersync and Stylelint)
- Web Browser
- CLI
- IDE

### Dependencies

- Browsersync
- Stylelint

### Install

`git clone`

`npm install`

### Run Linter Test

`stylelint "*/*.css"`

> https://stylelint.io/user-guide/cli

### Autofix CSS Linting Errors

`stylelint "*/**.css" --fix`

### Deployment

`npm start`
